```
   ↓
4. يختار العميل من القائمة
   ↓
5. تظهر تفاصيل العميل والباقة تلقائياً
   ↓
6. يتم ملء المبلغ والعملة تلقائياً
   ↓
7. يختار طريقة الدفع
   ↓
8. يختار الصندوق/البنك (حسب الطريقة)
   ↓
9. (تحويل بنكي) يدخل رقم المرجع + يرفع الصورة
   ↓
10. يضيف ملاحظات (اختياري)
    ↓
11. يضغط "إضافة الدفعة"
    ↓
12. النظام يحفظ البيانات
    ↓
13. (إذا الحالة: مكتمل) ينشئ قيد محاسبي
    ↓
14. تظهر رسالة نجاح
    ↓
15. يتحدث الجدول تلقائياً
```

## 🗄️ البيانات (Database)

### حقول جدول saas_payments

#### الحقول الأساسية
```sql
id                  UUID PRIMARY KEY
payment_number      VARCHAR(50)         -- تلقائي: PAY-2601-00001
tenant_id           UUID                -- إلزامي
product_id          UUID                -- تلقائي من العميل
subscription_id     UUID                -- تلقائي من العميل
plan_id             UUID                -- من الاشتراك
```

#### المبالغ والعملة
```sql
amount              DECIMAL(12,2)       -- إلزامي
currency            VARCHAR(3)          -- USD, SAR, EUR, etc.
```

#### طريقة الدفع
```sql
payment_method      VARCHAR(30)         -- cash, bank_transfer, etc.
account_id          UUID                -- الصندوق/الحساب
bank_account_id     UUID                -- البنك
wallet_id           UUID                -- المحفظة
account_name        VARCHAR(100)        -- اسم الحساب
reference_number    VARCHAR(100)        -- رقم المرجع
```

#### المستندات
```sql
receipt_url         TEXT                -- رابط الصورة
```

#### الحالة والتواريخ
```sql
status              VARCHAR(20)         -- completed, pending, failed
collection_date     TIMESTAMPTZ         -- تاريخ الاستلام
notes               TEXT                -- ملاحظات
```
