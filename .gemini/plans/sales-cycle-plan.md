## 🎯 مسار الحالات (Stage Flow)

```
مسودة ──→ مؤكدة ──→ قيد التسليم ──→ مسلّمة (جزئي/كلي) ──→ مرحّلة
draft    confirmed   in_delivery     delivered/partial      posted
                                     
                    ┌─ partial_delivery (تسليم جزئي)
                    │   يبقى in_delivery حتى اكتمال
                    │
                    └─ full_delivery (تسليم كامل)
                        → delivered → posted + قيد محاسبي
```

### حماية الحالات:
| المرحلة | تعديل | حذف | تسليم | مرتجع |
|---------|-------|-----|-------|-------|
| `draft` | ✅ | ✅ | ❌ | ❌ |
| `confirmed` | ⚠️ (مع إعادة إشعار) | ❌ | ✅ | ❌ |
| `in_delivery` | ❌ | ❌ | ✅ | ❌ |
| `delivered` | ❌ | ❌ | ❌ | ✅ |
| `posted` | ❌ | ❌ | ❌ | ✅ |

```
│   │   ROLL-001 | 150 م | أبيض                   │    │
│   │   ROLL-002 | 150 م | أبيض                   │    │

│   └─────────────────────────────────────────────┘    │
│                                                      │
│ ▶ 📦 قطن مصري                                        │
│   المطلوب: 150 م | المسلّم: 0 م | 0 رولون            │
│   ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░                     │

├──────────────────────────────────────────────────────┤
│ الإجمالي: 2 رولون | 300 م من 600 م | 50%             │
│ [تسليم جزئي مسموح ✅]                                │
│ [إلغاء]                    [✅ تأكيد التسليم]         │
└──────────────────────────────────────────────────────┘
```

**الوظائف:**
- سطر مطوي لكل مادة → ينفتح لعرض الرولونات
- قسم علوي: الرولونات المطلوبة (من الفاتورة) — للمرجع فقط
- قسم سفلي: الرولونات المُسلّمة فعلياً — قابلة للحذف
- مسح QR/RFID/باركود → يضيف الرولون تلقائياً
- حماية: فقط رولونات `status: available` من المستودع المحدد
- دعم التسليم الجزئي (لا يشترط تسليم كامل الكمية)

### المرحلة 2: تزامن الحالات 🟡

- عند فتح إذن التسليم: `confirmed → in_delivery`
- عند تأكيد تسليم جزئي: يبقى `in_delivery` + يسجّل الرولونات المسلّمة
- عند اكتمال التسليم (100%): `in_delivery → delivered`
- عند الترحيل: `delivered → posted` + قيد محاسبي
- إضافة عمود `delivered_qty` في `sales_transaction_items`
