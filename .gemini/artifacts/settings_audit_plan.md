## 🚨 المشاكل المكتشفة

### مشكلة 1: الإعدادات مبعثرة
- كل قسم (محاسبة، مستودعات، مبيعات) له صفحة إعدادات منفصلة
- لا يوجد مكان واحد لرؤية كل الإعدادات

### مشكلة 2: بيانات الشركة بلا واجهة
- الحقول موجودة في DB (اسم، شعار، رقم ضريبي، عنوان...)
- **لا يوجد أي مكان في النظام** لتعديلها!
- هذه البيانات ضرورية للفواتير والتقارير

### مشكلة 3: الحسابات الافتراضية فارغة
- كل الحسابات الافتراضية (صندوق، بنك، إيرادات، مصروفات...) = غير محددة
- الفواتير والقيود المحاسبية لن تعمل بدونها

### مشكلة 4: ترقيم الفواتير غير موجود
- فقط ترقيم قيود اليومية (JE)
- لا يوجد ترقيم لـ: فواتير المبيعات، فواتير المشتريات، سندات القبض، سندات الصرف

### مشكلة 5: السنة المالية غير معرفة
- لا يمكن تسجيل أي عملية بدون سنة مالية فعالة

## 🎯 الخطة: صفحة إعدادات موحدة `/system-config`

### التبويبات المقترحة:

#### 📌 تبويب 1: بيانات المنشأة (Company Profile) ← **جديد**
```
- اسم الشركة (عربي / إنجليزي)
- الشعار (logo_url)
- الدولة (country_code)
- المدينة (city)
- العنوان (address)
- الهاتف / البريد / الموقع
- الرقم الضريبي (tax_number)
- السجل التجاري (commercial_register)
- النظام الضريبي (tax_system)
- اللغة الرئيسية
```
**المصدر:** جدول `companies` — يحتاج فقط لواجهة UPDATE

#### 📌 تبويب 2: المحاسبة (Accounting Settings) ← **نقل من** `/accounting/settings`
```
- العملات (base_currency, supported_currencies)
- الحسابات الافتراضية
- نسبة الضريبة
- السنوات المالية
- ترقيم القيود والفواتير
- إعدادات التعديل
```

#### 📌 تبويب 3: المستودعات (Warehouse Settings) ← **نقل من** تبويب المستودعات
```
- طريقة التكلفة
- إعدادات التصدير والحجز
- Dye Lot settings
- إعدادات الباركود
```

#### 📌 تبويب 4: المبيعات (Sales Settings) ← **نقل من** تبويب المبيعات
```
- سير العمل
- الحالات والانتقالات
- الأتمتة
```

#### 📌 تبويب 5: المشتريات (Purchases Settings) ← **نقل من** تبويب المشتريات
```
- سير العمل
- الحالات والانتقالات
```

#### 📌 تبويب 6: الأدوار والصلاحيات ← **الموجود حالياً**
```
- إدارة الأدوار
- إدارة المستخدمين
- صلاحيات الموارد
- قواعد الإخفاء
```

#### 📌 تبويب 7: الموديولات والأنظمة ← **الموجود (فارغ)**
```
- تفعيل/إلغاء الموديولات
- ZATCA / Zakat settings
```
