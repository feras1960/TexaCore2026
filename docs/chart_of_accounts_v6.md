## 🏗️ Functions المعتمدة

### `create_simple_chart(p_company_id uuid)` — V6
- **56 حساب** بترقيم موحد (1-5)
- العملة من `company_accounting_settings.base_currency`
- حسابات الصندوق/البنك بالعملة الأساسية (`1112`, `1122`) = العملة الدولية
- تشمل: صندوق، بنوك، عملاء، مخزون، أصول ثابتة، خصوم، حقوق ملكية، إيرادات، مصروفات

### `create_extended_chart(p_company_id uuid)` — V6
- **90 حساب** بترقيم موحد (1-5)
- تشمل كل ما في القياسية + توسعات:
  - صندوق النثرية (1113)
  - ذمم الجملة/التجزئة/أوراق القبض (1131-1133)
  - بضاعة في الطريق (115) — بالعملة الدولية
  - أصول ثابتة مُفصّلة (121-129)
  - موردين رئيسي/أخرى (2111-2112)
  - شركات شحن/تخليص/نقل/تأمين/لوجستيك (2121-2125)
  - شركات صرافة (2126, 21261)
  - خصوم طويلة الأجل + التزامات إيجار (221-222)
  - تكاليف مبيعات مُفصّلة (511-513)
  - مشتريات/مردودات/خصومات (521-523)
  - مصاريف شحن/جمركة/تأمين بحري (581-584)
  - مصروفات متنوعة 8 أنواع (591-598)

### `apply_chart_template_to_company(p_company_id, p_template_code)` — V6
- **نقطة الدخول الرسمية** لتطبيق القالب
- `'simple'` أو `'SIMPLE'` → يستدعي `create_simple_chart()`
- `'extended'` أو `'EXTENDED'` → يستدعي `create_extended_chart()`

### `upgrade_company_chart(p_company_id, p_target_chart_type)` — V5.1
- ترقية تدريجية من Simple إلى Extended
- يضيف الحسابات الناقصة فقط بدون مساس البيانات الموجودة

### `setup_chart_templates_for_tenant(p_tenant_id)` — V5.1
- يُنشئ سجلات `chart_templates` للمستأجر الجديد
- قالبين فقط: SIMPLE + EXTENDED

### `create_summary_accounts_for_company(p_company_id)`
- يُنشئ حسابات تجميعية تلقائياً

## 💱 منطق العملات

### مصدر العملة:
```sql
SELECT COALESCE(cas.base_currency, c.default_currency, 'USD') -- العملة المحلية
       COALESCE(cas.default_international_purchase_currency, 'USD') -- العملة الدولية
FROM companies c
LEFT JOIN company_accounting_settings cas ON cas.company_id = c.id
```

### توزيع العملات على الحسابات:
| الحسابات | العملة |
|----------|--------|
| أغلب الحسابات | `base_currency` (المحلية) |
| صندوق عملة أساسية (1112) | `intl_currency` (الدولية) |
| بنك عملة أساسية (1122) | `intl_currency` |
| بضاعة في الطريق (115) | `intl_currency` |
| شحن بحري (2121) | `intl_currency` |
| شركات تأمين (2124) | `intl_currency` |
| مصاريف الشحن (581) | `intl_currency` |
| تأمين بحري وشحن (583) | `intl_currency` |
