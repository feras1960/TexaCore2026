## 🔄 سير العمل

### **إنشاء مادة بسيطة**:
```
1. اضغط "مادة جديدة"
2. اختر التصنيف → كود تلقائي (FAB-0001)
3. املأ الاسم والوصف
4. اختر الوحدة (متر/ياردة/كجم)
5. (اختياري) فعّل الجرد بالرولون
6. احفظ ✅
```

### **إنشاء مادة بمتغيرات ألوان**:
```
1. اضغط "مادة جديدة"
2. املأ المعلومات الأساسية
3. انتقل إلى تبويب "المتغيرات"
4. فعّل "تفعيل متغيرات الألوان"
5. اختر الألوان (مثلاً: أحمر، أزرق، أخضر)
6. راجع ملخص الألوان:
   - FAB-0001-أحم
   - FAB-0001-أزر
   - FAB-0001-أخض
7. احفظ ✅
8. سيتم إنشاء 3 مواد منفصلة
```

### **إنشاء مادة بمتغيرات رسمات**:
```
1. اضغط "مادة جديدة"
2. املأ المعلومات الأساسية
3. انتقل إلى تبويب "المتغيرات"
4. فعّل "يحتوي على رسمات"
5. احفظ المادة الأساسية
6. من تبويب "الرسمات":
   - أضف رسمة جديدة
   - ارفع صورة
   - حدد الألوان
7. احفظ ✅
```

## ⚠️ ملاحظات مهمة

### **1. الرولون ليس وحدة أساسية**
- ❌ لا تستخدم "رولون" كوحدة أساسية
- ✅ استخدم متر/ياردة/كجم/قطعة
- ✅ فعّل "الجرد بالرولون" للإحصاء

### **2. المتغيرات تنشئ مواد منفصلة**
- كل متغير = مادة منفصلة
- كل مادة لها كود فريد
- كل مادة لها مخزون منفصل

### **3. الرسمات تُضاف بعد الحفظ**
- احفظ المادة الأساسية أولاً
- ثم أضف الرسمات من التبويب
- كل رسمة يمكن أن يكون لها ألوان متعددة

### **4. التبويب يظهر فقط في Create/Edit**
- في وضع "إنشاء": يظهر تبويب المتغيرات
- في وضع "تعديل": يظهر تبويب المتغيرات
- في وضع "عرض": لا يظهر (لأن المتغيرات تم إنشاؤها)
