# 🔐 نظام الصلاحيات والأدوار الشامل
## Comprehensive RBAC System with Clear Hierarchy

---

## ⚠️ تنبيه مهم: فصل المستويات

هذا النظام يفصل **بشكل واضح** بين:

| المستوى | الدور | الوصف |
|---------|-------|-------|
| 🔴 **المنصة (SaaS)** | `super_admin` | أنت فقط - تدير كل شيء |
| 🟠 **المستأجر** | `tenant_admin` | يدير كل شركات المستأجر |
| 🟡 **الشركة** | `company_owner` | مالك الشركة |
| 🟡 **الشركة** | `company_admin` | مدير الشركة |
| 🟢 **الفرع** | `branch_manager` | يدير فرعه فقط |
| 🔵 **العمليات** | `accountant`, `cashier`, etc. | أدوار تشغيلية |

---

## 📊 الهيكل الهرمي الكامل

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    🔴 المستوى 1: المنصة (Platform Level)                     │
├─────────────────────────────────────────────────────────────────────────────┤
│  super_admin      → مدير المنصة الكامل (أنت) - يرى ويدير كل شيء            │
│  support_senior   → دعم فني أول - يرى كل المستأجرين (قراءة)               │
│  support          → دعم فني عادي - قراءة محدودة                            │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                   🟠 المستوى 2: المستأجر (Tenant Level)                      │
├─────────────────────────────────────────────────────────────────────────────┤
│  tenant_admin     → مدير المستأجر - يدير كل شركات المستأجر                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                   🟡 المستوى 3: الشركة (Company Level)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│  company_owner    → مالك الشركة - كل الصلاحيات داخل الشركة                  │
│  company_admin    → مدير الشركة - إدارة الشركة (بدون حذف)                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                     🟢 المستوى 4: الفرع (Branch Level)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│  branch_manager   → مدير الفرع - يدير فرعه فقط                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                 🔵 المستوى 5: العمليات (Operations Level)                    │
├─────────────────────────────────────────────────────────────────────────────┤
│  accountant       → محاسب - كل المحاسبة                                    │
│  cashier          → أمين صندوق - صندوقه المُعيّن فقط                       │
│  warehouse_keeper → أمين مستودع - مستودعه المُعيّن فقط                     │
│  sales_rep        → مندوب مبيعات - مبيعاته فقط                            │
│  purchaser        → مسؤول مشتريات - كل المشتريات                          │
│  employee         → موظف عادي - قراءة فقط                                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                       ⚪ دور خاص (Special Role)                              │
├─────────────────────────────────────────────────────────────────────────────┤
│  auditor          → مدقق - قراءة + سجلات التعديل (لا يمكنه التعديل)        │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 🔴 المستوى 1: أدوار المنصة (Platform)

### `super_admin` - مدير المنصة 👑
```json
{
  "all": true,
  "level": "platform"
}
```
- ✅ أنت فقط (feras1960@gmail.com)
- ✅ يرى ويدير كل المستأجرين
- ✅ يرى ويدير كل الشركات
- ✅ إدارة الباقات والاشتراكات
- ⚠️ **لا تُعطِ هذا الدور لأي مستخدم آخر!**

### `support_senior` - دعم فني أول
```json
{
  "level": "platform",
  "view_all_tenants": true,
  "read_only": true
}
```
- ✅ يرى كل المستأجرين
- ❌ لا يمكنه التعديل على البيانات

### `support` - دعم فني عادي
```json
{
  "level": "platform",
  "read_only": true,
  "limited_view": true
}
```

---

## 🟠 المستوى 2: أدوار المستأجر (Tenant)

### `tenant_admin` - مدير المستأجر
```json
{
  "level": "tenant",
  "scope": "all_companies",
  "companies": {"create": true, "read": true, "update": true, "delete": true},
  "users": {"create": true, "read": true, "update": true, "delete": true}
}
```
- ✅ يدير كل شركات المستأجر
- ✅ يُنشئ ويحذف الشركات
- ✅ يُنشئ ويدير المستخدمين
- ✅ يتحكم في الفوترة والموديولات

---

## 🟡 المستوى 3: أدوار الشركة (Company)

### `company_owner` - مالك الشركة
```json
{
  "level": "company",
  "scope": "company",
  "all_in_company": true,
  "users": {"delete": true}
}
```
- ✅ كل الصلاحيات داخل الشركة
- ✅ يحذف المستخدمين
- ✅ يحذف الفروع
- ❌ لا يرى شركات أخرى

### `company_admin` - مدير الشركة
```json
{
  "level": "company",
  "scope": "company",
  "users": {"delete": false}
}
```
- ✅ إدارة الشركة
- ❌ لا يمكنه حذف المستخدمين
- ❌ لا يمكنه حذف الفروع

---

## 🟢 المستوى 4: أدوار الفرع (Branch)

### `branch_manager` - مدير الفرع
```json
{
  "level": "branch",
  "scope": "assigned_branches",
  "funds": {"scope": "branch"},
  "warehouses": {"scope": "branch"}
}
```
- ✅ يدير فرعه فقط
- ✅ يرى صناديق ومستودعات فرعه
- ❌ لا يرى فروع أخرى

---

## 🔵 المستوى 5: أدوار العمليات (Operations)

### `accountant` - محاسب
- ✅ كل العمليات المحاسبية
- ✅ إنشاء وترحيل القيود
- ✅ إغلاق الفترات
- 👁️ قراءة المبيعات والمشتريات

### `cashier` - أمين صندوق
- 🔒 يرى صندوقه المُعيّن فقط
- ✅ إنشاء سندات القبض والصرف
- ❌ لا يرى صناديق أخرى

### `warehouse_keeper` - أمين مستودع
- 🔒 يرى مستودعه المُعيّن فقط
- ✅ استلام وصرف البضاعة
- ✅ جرد المخزون
- ❌ لا يرى مستودعات أخرى

### `sales_rep` - مندوب مبيعات
- 🔒 يرى مبيعاته فقط
- ✅ إنشاء عملاء وعروض أسعار
- ❌ لا يرى مبيعات الآخرين

### `purchaser` - مسؤول مشتريات
- ✅ كل عمليات الشراء
- ✅ إدارة الموردين

### `employee` - موظف عادي
- 👁️ قراءة فقط
- 🔒 يرى بياناته فقط

---

## ⚪ دور خاص: المدقق

### `auditor` - مدقق/مراجع
```json
{
  "level": "special",
  "read_only": true,
  "audit_logs": {"read": true},
  "document_edit_history": {"read": true}
}
```
- 👁️ قراءة كل شيء في الشركة
- ✅ وصول لسجلات التعديل
- ❌ لا يمكنه التعديل على أي شيء

## 📁 الجداول الجديدة

### `user_fund_permissions`
| الحقل | النوع | الوصف |
|-------|------|-------|
| `user_id` | UUID | المستخدم |
| `fund_account_id` | UUID | حساب الصندوق |
| `can_view` | BOOLEAN | عرض |
| `can_deposit` | BOOLEAN | إيداع |
| `can_withdraw` | BOOLEAN | سحب |
| `daily_limit` | DECIMAL | الحد اليومي |
| `is_primary` | BOOLEAN | الصندوق الرئيسي |

### `user_warehouse_permissions`
| الحقل | النوع | الوصف |
|-------|------|-------|
| `user_id` | UUID | المستخدم |
| `warehouse_id` | UUID | المستودع |
| `can_view` | BOOLEAN | عرض |
| `can_receive` | BOOLEAN | استلام |
| `can_issue` | BOOLEAN | صرف |
| `is_keeper` | BOOLEAN | أمين المستودع |

### `user_branch_permissions`
| الحقل | النوع | الوصف |
|-------|------|-------|
| `user_id` | UUID | المستخدم |
| `branch_id` | UUID | الفرع |
| `can_access` | BOOLEAN | الوصول |
| `can_manage` | BOOLEAN | الإدارة |
| `is_primary` | BOOLEAN | الفرع الرئيسي |

## 🔧 الدوال المساعدة

| الدالة | الغرض |
|--------|-------|
| `is_super_admin()` | هل هو مدير المنصة؟ |
| `is_tenant_admin()` | هل هو مدير المستأجر؟ |
| `is_company_admin()` | هل هو مالك/مدير الشركة؟ |
| `get_user_role_level()` | الحصول على مستوى الدور |
| `can_access_fund()` | التحقق من صلاحية الصندوق |
| `can_access_warehouse()` | التحقق من صلاحية المستودع |
| `get_user_full_permissions()` | جلب كل صلاحيات المستخدم |
