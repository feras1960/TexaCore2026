# 🚀 الميزات المخططة

## 1. نظام تخصيص الشيت بدون كود (No-Code Sheet Customizer)

**الحالة:** 📋 مخطط

**الوصف:** تمكين المستخدم من تخصيص واجهات التفاصيل (Detail Sheets) بدون برمجة.

**المتطلبات:**
- [ ] سحب وإفلات الحقول
- [ ] إضافة حقول مخصصة (نص، رقم، تاريخ، شارة...)
- [ ] تعدد اللغات للتسميات
- [ ] شروط الظهور
- [ ] تنسيق الحقول
- [ ] حفظ واستعادة التخصيصات

**الجداول المطلوبة:**
```sql
sheet_customizations  -- التخصيصات الرئيسية
custom_fields         -- الحقول المخصصة
field_translations    -- ترجمات الحقول
customization_history -- سجل التغييرات
```

**المكتبات المطلوبة:**
```json
{
  "@dnd-kit/core": "^6.1.0",
  "@dnd-kit/sortable": "^8.0.0"
}
```

**خطة التنفيذ:**
| المرحلة | المهام | المدة المتوقعة |
|---------|--------|----------------|
| 1 | البنية التحتية (DB + Service) | أسبوع 1 |
| 2 | المكونات الأساسية (Drag & Drop) | أسبوع 2 |
| 3 | محرر الحقول والترجمات | أسبوع 3 |
| 4 | التكامل والاختبار | أسبوع 4 |
