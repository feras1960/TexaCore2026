
|-------------|---------------------|
| قيمة البضاعة FOB | 70-80% |
| الشحن البحري | 8-12% |
| الجمارك | 5-10% |
| التأمين | 1-2% |
| مصاريف التخليص | 2-4% |
| النقل الداخلي | 1-3% |

## 🔒 الأمان والصلاحيات

### 5.1 Row Level Security (RLS)

جميع الـ 24 جدول مُفعّل عليها RLS:

```sql
-- نموذج السياسة المُطبقة
CREATE POLICY tenant_isolation ON table_name
    FOR ALL 
    USING (
        tenant_id IN (
            SELECT tenant_id FROM tenant_users 
            WHERE user_id = auth.uid()
        )
    );
```
