## الخطوة 1️⃣: إنشاء جدول tenant_subscriptions

### أ. افتح Supabase SQL Editor
1. اذهب إلى `https://supabase.com`
2. سجل الدخول
3. اختر مشروعك
4. من القائمة: **SQL Editor**

### ب. نسخ ولصق الكود
1. افتح الملف:
```
supabase/migrations/STEP_56B_create_tenant_subscriptions.sql
```

2. انسخ **كل** محتوى الملف (`Ctrl+A` ثم `Ctrl+C`)

3. في Supabase SQL Editor:
   - اضغط **"New query"**
   - الصق الكود (`Ctrl+V`)

### ج. التنفيذ
1. اضغط **"Run"** (أو `Ctrl+Enter`)
2. انتظر بضع ثوانٍ ⏳

### د. التحقق من النجاح
يجب أن ترى:
```
✅ جدول tenant_subscriptions تم إنشاؤه بنجاح
✅ Indexes تم إنشاؤها
✅ Triggers تم إنشاؤها
✅ RLS Policies تم إنشاؤها

📝 الآن يمكنك تنفيذ STEP_57
```

**إذا رأيت هذه الرسائل → انتقل للخطوة 2️⃣**

## الخطوة 2️⃣: إنشاء نظام الدفعات (STEP_57)

### أ. استعلام جديد
في Supabase SQL Editor:
- اضغط **"New query"** (استعلام جديد)

### ب. نسخ ولصق
1. افتح الملف:
```
supabase/migrations/STEP_57_saas_payments_infrastructure.sql
```

2. انسخ **كل** محتوى الملف

3. الصق في الاستعلام الجديد

### ج. التنفيذ
1. اضغط **"Run"**
2. انتظر... (قد يستغرق 10-20 ثانية) ⏳

### د. التحقق من النجاح
يجب أن ترى:
```
✅ جدول saas_payments تم إنشاؤه
✅ Functions تم إنشاؤها (5 دوال)
✅ Triggers تم إنشاؤها
✅ RLS Policies تم إنشاؤها

إذا كان هناك tenants:
✅ دفعة 1: $299 - تحويل بنكي
✅ دفعة 2: $299 - نقدي
✅ دفعة 3: $299 - بطاقة ائتمان
📊 إجمالي الدفعات: $897
```

**إذا لم يكن هناك tenants:**
```
⚠️  لا يوجد tenants - تخطي إضافة الدفعات
```
هذا طبيعي! يمكنك إضافة tenants لاحقاً.

## الخطوة 4️⃣: اختبار بسيط

في SQL Editor، جرّب:

```sql
-- 1. تحقق من جدول الاشتراكات
SELECT COUNT(*) as subscriptions_count 
FROM tenant_subscriptions;

-- 2. تحقق من جدول الدفعات
SELECT COUNT(*) as payments_count 
FROM saas_payments;

-- 3. إجمالي الإيرادات
SELECT get_total_revenue('USD') as total_revenue;

```

## ❓ أسئلة شائعة

### س1: لماذا لم يكن جدول tenant_subscriptions موجوداً؟
**ج:** كان مفترضاً في migrations سابقة، لكن لم يتم إنشاؤه. الآن أنشأناه.

### س2: إذا لم تُضف البيانات التجريبية، ماذا أفعل؟
**ج:** طبيعي إذا لم يكن هناك tenants. يمكنك:
1. إنشاء tenant من Dashboard
2. أو إضافة بيانات يدوياً (سأعطيك السكريبت لاحقاً)

### س3: هل يمكنني إعادة التنفيذ؟
**ج:** نعم، السكريبتات آمنة. تستخدم `IF NOT EXISTS` لتجنب الأخطاء.
