### الخطوة 4: إضافة دفعة جديدة

1. اضغط على زر **"إضافة دفعة"** (Add Payment)
2. ابحث عن العميل بالاسم أو الكود
3. اختر العميل من القائمة
4. ستظهر تفاصيل العميل والباقة تلقائياً
5. أدخل المبلغ والعملة (يتم ملؤها تلقائياً من سعر الباقة)
6. اختر تاريخ الاستلام (افتراضياً اليوم)
7. اختر طريقة الدفع:
   - **نقدي**: اختر الصندوق
   - **تحويل بنكي**: اختر البنك + أدخل رقم المرجع + ارفع الإشعار
   - **بطاقة ائتمان**
   - **محفظة رقمية**
   - **شيك**
8. اختر الحالة (مكتمل، معلق، فشل)
9. أضف ملاحظات (اختياري)
10. اضغط **"إضافة الدفعة"**

## ✅ اختبار النظام

### 1. التحقق من وجود عملاء

```sql
SELECT 
    t.id,
    t.name,
    t.code,
    sp.name as product_name,
    sub.status as subscription_status
FROM tenants t
LEFT JOIN saas_products sp ON t.product_id = sp.id
LEFT JOIN tenant_subscriptions sub ON t.id = sub.tenant_id
WHERE t.status IN ('active', 'trial')
LIMIT 10;
```

### 2. إضافة دفعة تجريبية

1. افتح النموذج
2. ابحث عن أي عميل
3. أدخل مبلغ 100
4. اختر "نقدي" كطريقة دفع
5. اختر أي صندوق
6. احفظ

### 3. التحقق من حفظ الدفعة

```sql
SELECT * FROM saas_payments 
ORDER BY created_at DESC 
LIMIT 5;
```

## 📊 إحصائيات

### عرض المدفوعات حسب الطريقة

```sql
SELECT 
    payment_method,
    COUNT(*) as total_payments,
    SUM(amount) as total_amount,
    currency
FROM saas_payments
WHERE status = 'completed'
GROUP BY payment_method, currency
ORDER BY total_amount DESC;
```

### عرض المدفوعات حسب العميل

```sql
SELECT 
    t.name as customer_name,
    COUNT(p.id) as payment_count,
    SUM(p.amount) as total_paid,
    p.currency
FROM saas_payments p
JOIN tenants t ON p.tenant_id = t.id
WHERE p.status = 'completed'
GROUP BY t.name, p.currency
ORDER BY total_paid DESC;
```
