## 🎯 مثال الاستخدام:

```
1. المستخدم يذهب لـ /saas/packages
2. يرى Tab "عرض جدولي" (افتراضي)

4. الجدول يعرض فقط باقات نيكسا كور
5. الإحصائيات تتحدث تلقائياً
6. يمكن النقر على أي صف لعرض التفاصيل
7. يمكن تصدير البيانات
8. يمكن التبديل لـ "عرض البطاقات" للعرض الأصلي
```
