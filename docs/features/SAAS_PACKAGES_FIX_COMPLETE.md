1. **إنشاء جدول `plan_modules`:**
```sql
CREATE TABLE plan_modules (
    id UUID PRIMARY KEY,
    plan_id UUID REFERENCES subscription_plans(id),
    module_id UUID REFERENCES system_modules(id),
    is_enabled BOOLEAN DEFAULT true,
    is_core BOOLEAN DEFAULT false, -- موديول أساسي (لا يمكن إزالته)
```

## 🔧 كيفية الاستخدام

### 1. إضافة موديول لباقة:

**الخطوات:**
1. افتح صفحة الباقات: `/saas/packages`
2. اضغط على باقة
3. انتقل إلى تبويب "Modules"
4. في قسم "الموديولات المتاحة"، اضغط "إضافة" على الموديول المطلوب
5. ✅ سيظهر Toast: "تم الحفظ بنجاح"
6. سيتم تحديث القائمة تلقائياً

**API Call:**
```typescript
await supabase
  .from('plan_modules')
  .insert({
    plan_id: 'xxx',
    module_id: 'yyy',
  });
```

---

### 2. إزالة موديول من باقة:

**الخطوات:**
1. في تبويب "Modules"
2. ابحث عن الموديول في قسم "الموديولات المفعلة"
3. اضغط زر الحذف (🗑️)
4. ✅ سيظهر Toast: "تم الحذف بنجاح"

**ملاحظة:** الموديولات الأساسية (Core) **لا يمكن** حذفها.

**API Call:**
```typescript
await supabase
  .from('plan_modules')
  .delete()
  .eq('id', 'plan_module_id');
```

---

### 3. تعديل حدود الباقة:

**الخطوات:**
1. انتقل إلى تبويب "Limits & Features"
2. اضغط زر "تعديل" (✏️)
3. غيّر القيم المطلوبة
   - للقيم العادية: أدخل رقم موجب
   - للقيم غير المحدودة: أدخل `-1` أو `0`
4. اضغط "حفظ" (💾)
5. ✅ سيظهر Toast: "تم التحديث بنجاح"

**API Call:**
```typescript
await supabase
  .from('subscription_plans')
  .update({
    max_users: 50,
    max_companies: 10,
    max_customers: 500,
    max_products: 5000,
    // ...
  })
  .eq('id', 'plan_id');
```

**القيم الخاصة:**
- `-1` = Unlimited (غير محدود)
- `0` = None/Disabled (معطل)
- `> 0` = العدد المحدد
