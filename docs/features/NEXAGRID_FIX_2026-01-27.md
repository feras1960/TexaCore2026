## 🔬 التشخيص

### المشكلة الأساسية: **تعارض CSS في AG Grid Container**

**الملف:** `src/components/shared/tables/NexaGrid.tsx`

**السطر 2056-2061 (قبل الإصلاح):**

```typescript
<div 
  className={cn('ag-theme-alpine w-full flex-1', isRTL && 'rtl-mode')}
  style={{ 
    height: typeof height === 'number' ? height : height,  // ❌ مشكلة: height بدون 'px'
    width: '100%',
    minWidth: 0,
  }}
>
```

### الأخطاء المكتشفة:

#### 1. ❌ `flex-1` + `height` معاً
- الـ container يستخدم `flex-1` (يأخذ كل المساحة المتبقية)
- في نفس الوقت `height` محدد = 500
- **النتيجة:** تعارض CSS يؤدي إلى height = 0 فعلياً!

#### 2. ❌ `height` بدون وحدة `px`
```typescript
height: typeof height === 'number' ? height : height
// إذا كان height = 500، النتيجة: height: 500 ❌
// يجب: height: "500px" ✅
```

#### 3. ✅ البيانات موجودة
- تم التحقق من `data` في `GRID_DOC_CONFIGS` - موجودة ✅
- تم التحقق من `columns` - معرّفة بشكل صحيح ✅
- AG Grid مثبت (v35.0.1) ✅
- CSS مستورد ✅

**الخلاصة:** المشكلة في CSS Layout، ليست في البيانات!

1. ✅ **أزلنا `flex-1`** من className

2. ✅ **أضفنا `px`** للـ height
   - تحويل من: `height: 500` → `height: "500px"`
   - CSS يحتاج وحدة قياس صريحة

## 🔧 التفاصيل التقنية

### لماذا كانت المشكلة تحدث؟

#### 1. Flexbox Conflict:
```css
/* Parent */
.parent {
  display: flex;
  flex-direction: column;
}

/* Child (AG Grid Container) */
.child {
  flex: 1;           /* يأخذ كل المساحة المتبقية */
  height: 500px;     /* محدد بـ 500px */
}

/* النتيجة: المتصفح يتجاهل أحدهما! */
```

#### 2. Invalid CSS Value:
```css
/* ❌ خطأ */
height: 500;   /* بدون وحدة قياس - يُتجاهل */

/* ✅ صحيح */
height: 500px; /* مع px */
height: 50vh;  /* أو vh */
height: 100%;  /* أو % */
```

### الحل في AG Grid:
```typescript
// عندما height محدد → لا تستخدم flex-1
// عندما تريد flex layout → لا تحدد height

// ✅ الخيار 1: Fixed Height (استخدمناه)
className="ag-theme-alpine w-full"
style={{ height: '500px' }}

// ✅ الخيار 2: Flex (لو أردنا flexible height)
className="ag-theme-alpine w-full flex-1"
style={{ minHeight: 0 }} // بدون height محدد
```

## 🚨 ملاحظات مهمة

### للمطورين:

1. **AG Grid يحتاج container بحجم محدد:**
   - يجب تحديد `height` صريحاً (px أو vh أو %)
   - أو استخدام flexbox بشكل صحيح

2. **لا تخلط flex-1 مع height محدد:**
   - إما `flex-1` (المساحة المتبقية)
   - أو `height: XXpx` (حجم ثابت)
   - **ليس الاثنين معاً!**

3. **دائماً أضف وحدة قياس:**
   - ✅ `height: "500px"`
   - ✅ `height: "100%"`
   - ❌ `height: 500`

4. **AG Grid CSS Classes:**
   - `ag-theme-alpine` - ضروري للتنسيق
   - `w-full` - عرض 100%
   - `rtl-mode` - للغة العربية
