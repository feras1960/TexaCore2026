🎯 القواعد الذهبية
───────────────────────────────────────────────────────────────────────────

1. Multi-Tenancy:
   ⚠️ دائماً أضف tenant_id و company_id في الاستعلامات!
   
   const { tenantId, companyId } = useAuth();
   supabase.from('products').select('*')
     .eq('tenant_id', tenantId)
     .eq('company_id', companyId)

2. الموديولات الديناميكية:
   ⚠️ دائماً تحقق من صلاحية الموديول قبل العرض!
   
   const { hasModule } = useModules();
   {hasModule('accounting') && <AccountingLink />}

3. الميزات الديناميكية:
   ⚠️ دائماً تحقق من صلاحية الميزة قبل عرض الأزرار!
   
   const { hasFeature } = useFeatures();
   {hasFeature('pdf_export') && <ExportButton />}

4. اللغات المتعددة:
   ⚠️ اعرض فقط اللغات المفعلة، وليس كل الـ 9!
   
   const { activeLanguages } = useLanguages();
   {activeLanguages.map(lang => <Input name={`name_${lang.code}`} />)}

5. مفاتيح الترجمة:
   ❌ أبداً: <Button>Save</Button>
   ✅ دائماً: <Button>{t('common.save')}</Button>

6. دعم RTL:
   ❌ أبداً: className="ml-4 text-left"
   ✅ دائماً: className="ms-4 text-start"
