🎯 GOLDEN RULES
───────────────────────────────────────────────────────────────────────────

1. Multi-Tenancy:
   ⚠️ ALWAYS include tenant_id and company_id in queries!
   
   const { tenantId, companyId } = useAuth();
   supabase.from('products').select('*')
     .eq('tenant_id', tenantId)
     .eq('company_id', companyId)

2. Dynamic Modules:
   ⚠️ ALWAYS check module access before showing features!
   
   const { hasModule } = useModules();
   {hasModule('accounting') && <AccountingLink />}

3. Dynamic Features:
   ⚠️ ALWAYS check feature access before showing buttons!
   
   const { hasFeature } = useFeatures();
   {hasFeature('pdf_export') && <ExportButton />}

4. Multi-Language:
   ⚠️ Show ONLY active languages, not all 9!
   
   const { activeLanguages } = useLanguages();
   {activeLanguages.map(lang => <Input name={`name_${lang.code}`} />)}

5. Translation Keys:
   ❌ NEVER: <Button>Save</Button>
   ✅ ALWAYS: <Button>{t('common.save')}</Button>

6. RTL Support:
   ❌ NEVER: className="ml-4 text-left"
   ✅ ALWAYS: className="ms-4 text-start"
