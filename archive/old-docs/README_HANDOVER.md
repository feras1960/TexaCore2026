## 🎯 القواعد الذهبية

قبل أن تبدأ، احفظ هذه القواعد:

1. ⚠️ **ALWAYS** include `tenant_id` + `company_id` in queries
2. ⚠️ **ALWAYS** check `hasModule()` before showing module
3. ⚠️ **ALWAYS** check `hasFeature()` before showing feature
4. ⚠️ **ALWAYS** show ONLY `activeLanguages` (not all 9!)
5. ⚠️ **NEVER** hard-code text → use `t('key.path')`
6. ⚠️ **NEVER** use `ml-4`, `text-left` → use `ms-4`, `text-start`
