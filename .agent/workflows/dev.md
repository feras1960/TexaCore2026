## Development Mode - وضع التطوير السريع

عند استخدام هذا الوورك فلو، يتم تنفيذ جميع الأوامر تلقائياً بدون طلب موافقة.

### الخطوات:
1. تنفيذ أوامر TypeScript check
2. تصفح الملفات والبحث
3. تشغيل أوامر npm
4. أي أمر آمن آخر مطلوب للتطوير

### ملاحظة:
- الأوامر المدمرة (حذف ملفات، تعديل قاعدة البيانات) ستظل تطلب موافقتك للأمان
- هذا الوورك فلو يُستخدم ضمنياً عند كتابة `/dev` في المحادثة
