```
fabric_materials.tax_rate (5%)
        ↓
container_items.tax_rate + allocated_tax + tax_per_unit
        ↓
  القيد المحاسبي: Dr 117 / Cr حساب الدفع (إجمالي)
        ↓
  عند البيع: وحدة البيع = وحدة المادة (متر/يارد/كغ)

```
