# 🚀 تحديث مسار الفاتورة وإصلاح الأخطاء
## تم إنجاز ما يلي:
1. **تصحيح سير العمل**: الفاتورة تنتقل الآن من `مسودة` -> `مؤكدة` -> `مرحّلة`.
2. **إصلاح الأزرار**: يظهر زر "حفظ وتأكيد" للمسودة، وزر "ترحيل" للمؤكدة فقط.
3. **حل مشكلة الحفظ**: تم إزالة الأعمدة غير الموجودة (`confirmed_at`) التي كانت تسبب فشل الحفظ.
4. **تحديث العناوين**: تظهر الآن "مشتريات مؤكدة" بدلاً من "مسودة مشتريات" بعد التأكيد.
5. **تصحيح الترجمة**: زر "حفظ وتأكيد" يظهر الآن بالعربية الصحيحة.

⚠️ **ملاحظة هامة**: يرجى إعادة تحميل الصفحة (`Cmd + R`) لتطبيق التغييرات.
