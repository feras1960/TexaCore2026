## 📐 هيكل القيد المحاسبي (المنطق الحالي)

### مثال: شحن بحري لمايرسك = 5,000 USD + ضريبة 15%

```
القيد:
─────────────────────────────────────────────────
مدين │ 5814 مايرسك - شحن بحري   │ 5,000 USD
مدين │ 1160 ضريبة مدخلات         │   750 USD
دائن │ 1143 بضاعة بالطريق        │ 5,750 USD
─────────────────────────────────────────────────
البيان: شحن بحري — مايرسك — كونتينر CNT-0001
```

### مثال بدون ضريبة: رسوم جمركية = 3,000 USD

```
القيد:
─────────────────────────────────────────────────
مدين │ 5821 رسوم جمركية          │ 3,000 USD
دائن │ 1143 بضاعة بالطريق        │ 3,000 USD
─────────────────────────────────────────────────
```
